## Basic syntax

Statments are terminated with ;
Statment consists of lexpr (assignment) (rexpr) ;
All "statements" are expressions
Brace language - blocks are delimited with { and }

## Typing
Language is dynamically, gradually typed. Derived from a root Any type which can coerce to any more specific type.
None is the never type
Classes are objects and objects are classes
